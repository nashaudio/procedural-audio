# Procedural Audio Library
Open-source library of procedural audio sound effects, developed in C++ (klang).

These sounds are free to use in personal and commercial projects, subject to attribution per the licence (Klang Open Licence v1.0).

Please submit new or improved sounds to procedural@nash.audio. Please also let the project know how the sounds are being used.

## List of sounds

VEHICLES
- Toy Boat Engine
- Basic Car Engine
- Advanced Car Engine (Mini)
- Jet (Harrier)
- Bicycle

NATURE
- Rain (work-in-progress)
